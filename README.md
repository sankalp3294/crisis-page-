# crisis-link -
CrisisLink is an AI-powered emergency reporting platform. Users submit incidents with location and details. Gemini AI classifies urgency, then staff monitor and resolve issues through a live dashboard.
features offered by project 
• Incident report form
• Gemini severity analysis
• Staff login panel
• Status updates
• Incident history
• Cloud deploy ready
tech stacj of the project 
• HTML • CSS • JavaScript
• Python Flask
• SQLite
• Google Gemini 2.0 Flash API
